# Binge Social Media Platform - Design Guide

## Design Philosophy

**Color Palette**: 
- Primary: #2a0a3d (Deep purple)
- Lighter shade: #4a1a6d (For text and accents)
- Background: #0a0a0a (Dark)
- Text: #ffffff (White for contrast)
- Accent: #6b46c1 (Bright purple for highlights)

**Typography**:
- Display Font: "Inter" - Modern, clean sans-serif for headings
- Body Font: "Inter" - Consistent typography throughout
- Font weights: 300 (light), 400 (regular), 600 (semibold), 700 (bold)

**Visual Language**:
- Modern, sophisticated dark theme
- Clean geometric shapes and smooth animations
- Emphasis on user engagement and community building
- Professional yet approachable aesthetic

## Visual Effects

**Used Libraries**:
- Anime.js for smooth animations and transitions
- ECharts.js for the diversity score gauge visualization
- Pixi.js for background particle effects
- Typed.js for dynamic text animations
- Splitting.js for text reveal effects

**Effects**:
- Typewriter animation for hero text
- Smooth hover transitions on buttons and cards
- Particle background animation
- Circular menu with smooth rotation
- Diversity score gauge with animated needle
- Smooth page transitions

**Animation Styling**:
- Subtle fade-ins and slide-ups
- Smooth color transitions
- Gentle pulsing effects for interactive elements
- Responsive hover states with scale transforms

**Header Effect**:
- Clean navigation bar with glass morphism effect
- Smooth scrolling between sections
- Active state indicators for current page

**Background**:
- Consistent dark gradient background (#0a0a0a to #1a0a2a)
- Subtle particle system for visual interest
- No section-specific background colors to maintain consistency
